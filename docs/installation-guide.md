# Floaty 설치 가이드

Floaty는 부동(floating) 데스크톱 타이머 위젯입니다. 이 가이드는 macOS와 Windows에서 Floaty를 설치하고 실행하는 방법을 단계별로 설명합니다.

## 개요

Floaty는 다음 파일 형식으로 제공됩니다:

| 플랫폼 | 파일 | 아키텍처 | 설명 |
|--------|------|----------|------|
| macOS | `Floaty-0.1.0-mac.dmg` | Apple Silicon (arm64) | DMG 설치 프로그램 |
| macOS | `Floaty-0.1.0-arm64-mac.zip` | Apple Silicon (arm64) | ZIP 압축 파일 |
| Windows | `Floaty-0.1.0-win.exe` | x64 (Intel/AMD) | NSIS 설치 프로그램 |
| Windows | `Floaty-0.1.0-portable.exe` | x64 (Intel/AMD) | 포터블 (설치 불필요) |

**주의**: 이 앱은 코드 서명이 되어 있지 않습니다. 따라서 처음 실행할 때 보안 경고가 나타날 수 있습니다. 이는 정상이며 아래 지침을 따라 무시할 수 있습니다.

---

## macOS 설치 방법

### 방법 1: DMG 파일 사용 (권장)

1. GitHub Releases 페이지에서 `Floaty-0.1.0-mac.dmg` 파일을 다운로드합니다.
2. 다운로드된 DMG 파일을 더블클릭하여 엽니다.
3. 마운트된 폴더에 보이는 Floaty 앱을 **Applications 폴더**로 드래그합니다.
4. 설치가 완료되면 DMG 파일을 마운트 해제합니다 (창의 Eject 버튼 클릭).
5. Applications 폴더에서 Floaty를 찾아 실행합니다.

### 방법 2: ZIP 파일 사용

1. GitHub Releases 페이지에서 `Floaty-0.1.0-arm64-mac.zip` 파일을 다운로드합니다.
2. 다운로드된 ZIP 파일을 더블클릭하여 자동 압축 해제됩니다.
3. 압축이 해제된 `Floaty.app`을 **Applications 폴더**로 이동합니다.
4. Applications 폴더에서 Floaty를 찾아 실행합니다.

### macOS 보안 경고 해결 (중요!)

이 앱은 코드 서명이 되어 있지 않아 macOS의 Gatekeeper 보안 기능이 첫 실행을 차단합니다. 다음 방법 중 하나를 사용하여 해결할 수 있습니다:

#### 방법 1: Finder를 통한 실행 (가장 간단)

1. **Finder**를 열고 **Applications 폴더**로 이동합니다.
2. **Floaty.app**을 찾습니다.
3. **Floaty.app을 우클릭** (또는 Control+클릭)합니다.
4. 메뉴에서 **"열기"**를 선택합니다.
5. 나타나는 대화상자에서 **"열기"** 버튼을 클릭합니다.

이 방법으로 처음 한 번만 실행 후, 다음부터는 일반적으로 실행됩니다.

#### 방법 2: 시스템 설정에서 허용

1. **시스템 설정**을 엽니다 (Apple 메뉴 > 시스템 설정).
2. **개인정보 보호 및 보안**으로 이동합니다.
3. 창의 하단으로 스크롤합니다.
4. "**다음 앱이 차단되었습니다**" 섹션을 찾습니다.
5. Floaty 옆의 **"확인 없이 열기"** 버튼을 클릭합니다.

#### 방법 3: 터미널을 이용한 고급 방법

터미널에 다음 명령어를 입력하고 Enter 키를 누릅니다:

```bash
xattr -cr /Applications/Floaty.app
```

그 후 Applications 폴더에서 Floaty.app을 더블클릭하여 실행합니다.

## Windows 설치 방법

### 방법 1: 설치 프로그램 사용 (권장)

1. GitHub Releases 페이지에서 `Floaty-0.1.0-win.exe` 파일을 다운로드합니다.
2. 다운로드된 `.exe` 파일을 더블클릭합니다.
3. Windows SmartScreen 경고가 나타나면 **"추가 정보"**를 클릭한 후 **"실행"**을 클릭합니다 (아래 참조).
4. 설치 마법사를 따릅니다:
   - 설치 위치를 선택합니다 (기본값 사용 권장: `C:\Program Files\Floaty`).
   - **"설치"** 버튼을 클릭합니다.
5. 설치가 완료되면 **"마침"**을 클릭합니다.
6. 설치 후 바탕화면의 Floaty 바로가기나 시작 메뉴에서 앱을 실행합니다.

### 방법 2: 포터블 버전 사용 (설치 불필요)

1. GitHub Releases 페이지에서 `Floaty-0.1.0-portable.exe` 파일을 다운로드합니다.
2. 원하는 폴더 (예: 문서 폴더, 바탕화면)에 파일을 저장합니다.
3. 파일을 더블클릭하여 바로 실행합니다.
4. Windows SmartScreen 경고가 나타나면 **"추가 정보"**를 클릭한 후 **"실행"**을 클릭합니다 (아래 참조).

포터블 버전은 설치가 필요 없으므로, 다른 컴퓨터로 옮기거나 삭제할 때 매우 편리합니다.

### Windows SmartScreen 경고 해결 (중요!)

포터블 또는 설치 프로그램 실행 시 다음과 같은 보안 경고가 나타날 수 있습니다:

> "Windows가 PC를 보호했습니다"
> 또는
> "알 수 없는 게시자 - Windows가 이 파일의 실행을 중지했습니다"

이는 앱이 코드 서명되지 않았기 때문입니다. 다음 단계로 실행할 수 있습니다:

#### 단계 1: 보안 경고 화면에서

1. **"추가 정보"** 또는 **"자세히"** 링크를 클릭합니다.
2. **"실행"** 버튼이 나타나면 이를 클릭합니다.

#### 단계 2: "알 수 없는 게시자" 경고가 계속 나타나는 경우

다음 방법을 시도하세요:

1. 파일을 우클릭합니다.
2. **"속성"**을 선택합니다.
3. **"일반"** 탭에서 하단의 **"차단 해제"** 체크박스에 체크를 합니다.
4. **"적용"** 및 **"확인"**을 클릭합니다.
5. 이제 파일을 더블클릭하여 실행할 수 있습니다.

## 앱 제거 (삭제) 방법

### macOS에서 제거

**방법 1: Finder 사용**
1. **Finder**를 열고 **Applications 폴더**로 이동합니다.
2. **Floaty.app**을 선택합니다.
3. **휴지통**으로 드래그하거나, 우클릭하여 **"휴지통으로 이동"**을 선택합니다.
4. 휴지통을 비웁니다 (선택사항, 바로 제거하려면).

**방법 2: Launchpad 사용**
1. **Launchpad**를 엽니다.
2. Floaty 아이콘을 길게 누르면 제거 옵션이 나타납니다.
3. **"제거"** 또는 **"X"** 버튼을 클릭합니다.

### Windows에서 제거

**설치 프로그램으로 설치한 경우:**
1. **설정** > **앱** > **앱 및 기능** (또는 **프로그램 제거**)을 엽니다.
2. 목록에서 **Floaty**를 찾습니다.
3. Floaty를 선택하고 **"제거"** 버튼을 클릭합니다.
4. 제거 확인 창이 나타나면 **"예"**를 클릭합니다.

**포터블 버전을 사용한 경우:**
- 파일 탐색기에서 저장된 `Floaty-0.1.0-portable.exe` 파일을 찾아 삭제하면 됩니다. 레지스트리 변경이 없으므로 파일 삭제만으로 완전히 제거됩니다.

## 시스템 요구사항

### macOS
- **최소 운영 체제**: macOS 10.15 (Catalina) 이상
- **필수 아키텍처**: Apple Silicon (M1, M2, M3, M4 등)
- **권장 사항**: 최신 macOS 버전 권장

### Windows
- **최소 운영 체제**: Windows 10 이상
- **아키텍처**: x64 (Intel/AMD 64-bit). ARM64 Windows는 현재 미지원.
- **권장 사항**: Windows 11 권장

## 문제 해결

### macOS 관련 문제

#### 문제: "손상되었기 때문에 열 수 없습니다" 메시지

Floaty를 실행할 때 다음 메시지가 나타나는 경우:

> "Floaty" 앱이 손상되었거나 신뢰할 수 없는 개발자가 만들어 열 수 없습니다.

**해결 방법:**

터미널을 열고 다음 명령어를 입력합니다:

```bash
xattr -cr /Applications/Floaty.app
```

명령어 실행 후 Floaty.app을 다시 실행해봅니다.

#### 문제: "Permission denied" (권한 거부) 오류

**해결 방법:**

1. 터미널을 열고 다음을 입력합니다:
   ```bash
   chmod +x /Applications/Floaty.app/Contents/MacOS/Floaty
   ```
2. Floaty.app을 다시 실행합니다.

#### 문제: 앱이 시작 후 바로 종료됨

**해결 방법:**

1. **시스템 설정** > **개인정보 보호 및 보안**을 확인합니다.
2. "**이 앱이 차단되었습니다**" 섹션에서 Floaty를 찾고 **"확인 없이 열기"**를 클릭합니다.

### Windows 관련 문제

#### 문제: "알 수 없는 게시자" 경고가 반복됨

**해결 방법:**

1. 파일을 우클릭하고 **속성**을 선택합니다.
2. **"일반"** 탭을 찾고, 하단의 **"차단 해제"** 체크박스에 체크합니다.
3. **"적용"**을 클릭한 후 **"확인"**을 클릭합니다.
4. 파일을 다시 실행해봅니다.

#### 문제: 앱이 실행되지 않거나 크래시 발생

**해결 방법:**

1. 바이러스 백신 또는 방화벽 소프트웨어가 앱을 차단하고 있는지 확인합니다.
2. 바이러스 백신 설정에서 Floaty를 예외 목록에 추가합니다.
3. 필요한 경우 포터블 버전을 다시 다운로드하여 설치를 다시 시도합니다.

#### 문제: "VCRUNTIME.dll" 또는 ".NET Framework" 오류

**해결 방법:**

Windows 컴퓨터에 필요한 런타임 라이브러리가 없을 수 있습니다.

1. Microsoft Visual C++ 재배포 패키지를 설치합니다:
   - [Microsoft Visual C++ Redistributable](https://support.microsoft.com/en-us/help/2977003)에서 최신 버전을 다운로드합니다.
   - 설치 후 Floaty를 다시 시작합니다.

#### 문제: 파일 탐색기에서 EXE 파일 아이콘이 제대로 표시 안 됨

**해결 방법:**

이는 cosmetic 문제로, 앱 기능에는 영향을 주지 않습니다. 파일을 일반적으로 실행할 수 있습니다.

## 일반적인 질문 (FAQ)

**Q: Floaty를 설치하면 인터넷 연결이 필요한가요?**

A: 아니요. Floaty는 완전히 오프라인으로 작동하며, 인터넷 연결이 필요하지 않습니다.

**Q: Floaty가 개인 데이터를 수집하나요?**

A: 아니요. Floaty는 데이터를 수집하거나 외부로 전송하지 않습니다. 모든 기능은 로컬에서만 실행됩니다.

**Q: 다른 컴퓨터로 Floaty를 옮길 수 있나요?**

A: 네. 포터블 버전을 사용하거나, 설치된 버전의 경우 각 컴퓨터에서 다시 설치하면 됩니다.

**Q: 여러 Floaty 창을 동시에 실행할 수 있나요?**

A: 네. 앱을 여러 번 실행하면 여러 개의 타이머 창을 동시에 사용할 수 있습니다.

**Q: 업데이트는 어떻게 받나요?**

A: GitHub Releases 페이지에서 주기적으로 확인하고, 새 버전이 나오면 다시 다운로드하여 설치합니다.
